# 南草津皮フ科 スタッフ研修アプリ

## 概要
皮膚科専門医と共に安全な医療を提供するためのスタッフ研修Webアプリです。

## 機能
- 皮膚疾患TOP20の基礎知識・患者説明トレーニング
- 薬剤規格リスト（38薬剤・絞り込み検索対応）
- 禁忌事項（15件・重要度別カラー表示）
- 当院の美容施術・機器（16メニュー）
- スキンケア製品一覧（11製品）
- 業務チェックリスト（受付・クラーク・カウンセラー）
- 確認クイズ（15問・カテゴリ別・シャッフル出題）

## 技術スタック
- Next.js 16 (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- Zustand（状態管理）
- Vercelでホスティング

## 開発
```bash
pnpm install
pnpm run dev
```
